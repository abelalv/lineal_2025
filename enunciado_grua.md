
# 📌 Análisis de Fuerzas en una Grúa – Álgebra Lineal en \(\mathbb{R}^2\)

## **Contexto:**
Un ingeniero mecánico está diseñando una grúa móvil que debe levantar una carga de 500 kg. 
Dos cables están sosteniendo la carga desde diferentes ángulos, ejerciendo fuerzas **\( \mathbf{F}_1 \) y \( \mathbf{F}_2 \)**.
Se quiere analizar la eficiencia de estas fuerzas y verificar si el diseño es estable.

Las ecuaciones vectoriales de las fuerzas son:

\$
\mathbf{F}_1 =  (4, 3)
\$

\$
\mathbf{F}_2 =  (-2, 5)
\$

## **Tareas a Resolver:**

1. **Calcular la fuerza resultante total** sobre la carga. (Suma de vectores)
2. **Determinar la magnitud de cada fuerza individual.** (Norma de cada vector)
3. **Calcular el ángulo entre las dos fuerzas.** (Usando el producto punto)
4. **Interpretar si las fuerzas están en conflicto o ayudan a sostener la carga.** 
5. **Si la grúa necesita rediseñarse para mayor estabilidad, ¿qué modificación sugerirías en los vectores de fuerza?**

Este problema se resolverá utilizando **Python**.
